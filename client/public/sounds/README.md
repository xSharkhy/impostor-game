# Sound Effects

Place MP3 files here for game sound effects.

## Required Files

| File | Event | Description |
|------|-------|-------------|
| `join.mp3` | Player joins | Soft pop/chime |
| `leave.mp3` | Player leaves | Soft whoosh |
| `start.mp3` | Game starts | Dramatic reveal |
| `vote.mp3` | Vote cast | Click/tap |
| `eliminate.mp3` | Player eliminated | Dramatic sting |
| `victory.mp3` | Crew wins | Celebration fanfare |
| `defeat.mp3` | Impostor wins | Dark/sinister |
| `tick.mp3` | Countdown | Clock tick |
| `reveal.mp3` | Impostor revealed | Suspenseful reveal |
| `notification.mp3` | Generic alert | Soft notification |

## Recommended Sources

- [Mixkit](https://mixkit.co/free-sound-effects/) - Free sound effects
- [Freesound](https://freesound.org/) - CC-licensed sounds
- [Pixabay](https://pixabay.com/sound-effects/) - Royalty-free

## Format

- MP3 format
- Short duration (< 3 seconds)
- Normalized volume
